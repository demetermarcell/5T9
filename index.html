Game Description:

In this 5-minute game, you’ll be decoding messages written in T9, the old mobile texting system. Each message is a series of numbers, where each number represents a letter depending on how many times it’s pressed. For example, "Hello" is written as "4433555555666". Your task is to quickly translate the code into regular text before the timer runs out. The faster and more accurately you decode, the higher your score.

Good answer: +2p Wrong answer: -1p Skip Answer: 0p
